Sobre o Projeto

MEDICALSOFT é um sistema de gestão para clínicas, desenvolvido em C++ utilizando o framework Qt e o sistema de build CMake. O objetivo principal do projeto é fornecer uma solução eficiente e intuitiva para a gestão de pacientes, colaboradores, agendamentos, relatórios e estoque de produtos na clínica.

O sistema oferece funcionalidades como cadastro, edição e exclusão de pacientes e colaboradores, agendamento de sessões, controle de estoque e geração de relatórios detalhados sobre atendimentos. Utilizando a programação orientada a objetos e os recursos do Qt, o MEDICALSOFT garante modularidade, reutilização de código e manutenção facilitada.
